# Fonts

Add these font files from [Google Fonts](https://fonts.google.com/) (or your design kit):

**Quicksand**
- `Quicksand-Regular.ttf`
- `Quicksand-Medium.ttf`
- `Quicksand-SemiBold.ttf`
- `Quicksand-Bold.ttf`

**Lato**
- `Lato-Regular.ttf`
- `Lato-Italic.ttf`
- `Lato-Bold.ttf`

Place them in this folder so the app can load them. Font sizes and usage are defined in `lib/core/theme/app_typography.dart` and `app_theme.dart`.
